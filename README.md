# Meiju
## 数据来源字幕组

## 使用方法

```python

if __name__ == '__main__':

    update(['闪电侠','神盾局特工','少年谢尔顿','氪星'])

```
其中填入想看的美剧,以数组形式传入,没有存入sql的为更新的没看过的美剧,当出现更新时运行完之后会复制下载链接至剪辑版放进迅雷即可下载
